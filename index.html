Tutorial: Gerenciamento de Licenças Ambientais

1. Iniciar o Chat

Na tela inicial, clique no botão "Iniciar Chat" para começar a interagir com o sistema.

2. Registrar uma Licença

Digite "iniciar" no campo de entrada para começar o processo de registro. O sistema irá solicitar:

  • E-mail
  • Número da licença
  • Nome da licença
  • Nome do cliente
  • Local da licença
  • Data de "Protocolar até" (YYYY-MM-DD)
  • Data de vencimento (YYYY-MM-DD)

3. Adicionar ao Google Calendar

Após registrar a licença, você receberá um link para adicionar um lembrete ao Google Calendar. Clique no link para abrir o calendário.

4. Exportar Dados

Depois de registrar uma licença, clique em "Exportar Dados" no menu lateral para baixar os dados como um arquivo JSON.

5. Outras Funcionalidades

Explore outras opções no menu lateral:

  • Limpar Chat: Apaga o histórico do chat.
  • Modo Escuro: Alterna entre temas claro e escuro.
  • Créditos: Veja informações sobre os desenvolvedores.
