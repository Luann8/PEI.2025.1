Tutorial

1. Acessar o Chat

Na tela inicial, clique no botão " Iniciar Chat" para abrir a interface de conversa e começar a interagir com o sistema.

2. Registrar uma Licença

No chat, digite "iniciar" para começar o registro de uma licença ambiental. O sistema solicitará as seguintes informações:

  • Seu e-mail (ex.: exemplo@dominio.com)
  • Número da licença
  • Nome da licença
  • Nome do cliente
  • Local da licença
  • Data de "Protocolar até" (formato: AAAA-MM-DD, ex.: 2025-03-25)
  • Data de vencimento (formato: AAAA-MM-DD, ex.: 2025-04-01)

Responda a cada pergunta no chat para completar o registro.

3. Adicionar Lembrete ao Google Calendar

Após registrar a licença, o sistema fornecerá um link para adicionar um lembrete ao Google Calendar. Clique no link para criar um evento com os detalhes da licença.

4. Exportar Dados da Licença

Quando uma licença for registrada, a opção " Exportar Dados" aparecerá na barra lateral. Clique nela para baixar os dados da licença em formato JSON.

5. Usar a Barra Lateral

A barra lateral contém as seguintes funcionalidades:

  • Limpar Chat: Apaga todas as mensagens do chat e reinicia o processo.
  • Exportar Dados: Baixa os dados da licença registrada (aparece após o registro).
  • Tutorial: Abre este guia para aprender a usar o sistema.
  • Modo Escuro: Alterna entre os temas claro e escuro para melhor visualização.
  • Créditos: Exibe informações sobre os desenvolvedores do sistema.
